About

Swarm Mesh Manager — peer-to-peer seed networking for the COGS platform.

≡ Description

Complete mesh management for seamless peer-to-peer seed networks. Auto-discover peers on the local network via mDNS and UDP broadcast, securely configure mesh credentials, actively monitor per-peer health and latency, and view real-time vector-store sync status across the swarm. The integrated dashboard surfaces every seed in your hive constellation with live telemetry.
