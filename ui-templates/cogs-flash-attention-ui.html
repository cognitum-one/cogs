Flash Attention · Details

cognitum-one/cogs/src/cogs/flash-attention

Applies flash attention mechanisms to subcarrier groups for spatial focusing, improving signal-to-noise ratio in dense RF environments. The attention kernel operates on tiled CSI subcarrier groups and emits attention maps, focus zones, and SNR improvement deltas. Tile-fused softmax keeps the working set on-chip so the full attention pass fits in ESP32-S3 PSRAM with room to spare.
