📡 RuView WiFi DensePose · docs

Overview

Full-body 17-keypoint pose estimation through walls using WiFi CSI. Vital signs, activity recognition, and multi-person tracking. No cameras, no wearables, complete privacy preservation.

Events emitted

  • pose_keypoints{ person_id, kp[17][3] }
  • vital_signs{ person_id, hr_bpm, resp_per_min, hrv_ms }
  • activity{ person_id, label, confidence }
  • person_count{ total, per_room: {A,B,C} }
  • fall_detected{ person_id, vertical_velocity, ts }

Hardware

2× ESP32-S3 sensors recommended (TX + RX). CSI dense input. Place sensors on opposite walls of the monitored space, 3–6 m apart. PCB or dipole antennas both supported.

Install

# on the cognitum host cog install ruview-densepose cog calibrate ruview-densepose # vacate the room first cog start ruview-densepose

Subscribe (Python)

from cogs import subscribe async for evt in subscribe("ruview-densepose.pose_keypoints"): print(evt.person_id, evt.kp["nose"])

Privacy guarantees

No cameras, no audio, no biometric identifiers. Person IDs are session-scoped and never leave the device unless an alert webhook fires. Anonymous mode strips IDs entirely.
